# LearnHub

LearnHub is a full-stack MERN learning platform designed to help students learn web development, AI, programming, and modern technologies through interactive courses and practical projects.

## Features

- User Authentication
- Course Management
- Responsive Modern UI
- MERN Stack Architecture
- Interactive Learning Platform
- Student Dashboard
- Instructor Features

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB
- Tailwind CSS

## Installation

```bash
npm install
npm start
